# NeuroMatrix - Architecture Tuning

NeuroMatrix is a specialized platform providing theoretical mathematical projections for neural network architecture optimization and hyperparameter tuning.

## 🧠 Project Goal
To provide researchers and engineers with a high-fidelity visualization environment for exploring deep learning architectural decisions through local-first, privacy-preserving computation.

## 🛡️ Core Mandates
- **CLS Prevention**: Strict structural ID hooks (`#slot-lead-billboard`, `#slot-sidebar-skyscraper`) enforce layout stability for AdSense-compliant deployments.
- **Privacy First**: All architectural data processing occurs locally within the user's browser.
- **Theoretical Precision**: Models provide mathematical heuristics for educational and planning purposes.

## 🚀 Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Update `adsbygoogle` pub-id in `src/app/layout.tsx`.

## 📂 Architecture
- `/src/app`: Next.js App Router with optimized legal, about, and contact pages.
- `/src/components`: Specialized visualization components and `AdPlaceholder` with automated CLS ID mapping.
- `/src/lib`: Neural network mathematical models and shared utilities.
